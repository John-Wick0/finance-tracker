Important side note:

To turn your project into a git repo locally do this:
step 1 -> open cmd 
step 2 type -> git init 
step 3 type -> git add 
step 4 type -> git commit -m "Initial commit - your projects name"

Then to push to github: 
step 1: -> open github 
step 2: create a repo example(finance-tracker) 
step 3: open Bash or cmd 
step 4 type -> git remote add origin https://github.com/YOUR_USERNAME/finance-tracker.git 
step 5 type -> git branch -M main 
step 6 type -> push -u origin main
